# uestc_python

电子科技大学2020-2021python选修课



### 使用概述

1. **数据库迁移**

   1.1 `py manage.py makemigrations`

   1.2 `py manage.py migrate`

2. **创建超级用户**

   2.1 `py manage.py createsuperuser`

3. **启动服务**

   3.1 `py manage.py runserver`

4. **进入后台**（`127.0.0.1:8000/admin`），进行相关操作
